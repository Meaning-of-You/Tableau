##### 파일에 연결

로컬 컴퓨터에 존재하는 데이터를 연결할 때 사용

##### 서버에 연결

클라우드에 존재하는 데이터를 연결할 때 사용

- `자세히....`를 선택하면 나타나있는 것 외의 서버와 연동 가능
- 찾는 서버가 없다면 기타 데이터베이스(ODBC) 사용
